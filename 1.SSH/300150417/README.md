1. **Changer l’URL du dépôt distant**

   ```sh
   git remote set-url origin git@github.com:CollegeBoreal/INF1084-202-25A-04.git
   ```

2. **Vérifier la nouvelle configuration du dépôt distant**

   ```sh
   git remote --verbose
   ```

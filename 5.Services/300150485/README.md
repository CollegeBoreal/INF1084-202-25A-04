# TP5 - Services Active Directory (AD DS)

Ce dossier contient les scripts PowerShell utilisés pour gérer et vérifier les services Windows liés à Active Directory Domain Services (AD DS).

## Fichiers
- services1.ps1 → Liste et vérifie les services AD DS
- services2.ps1 → Affiche les logs NTDS et Netlogon
- services3.ps1 → Exporte les événements AD vers un fichier CSV
- services4.ps1 → Démontre l’arrêt/redémarrage d’un service AD
- images/ → contient les captures d’écran du TP5

# 300152410

TP : Partage de ressources, GPO (mapping de lecteur) et RDP via PowerShell.

## Objectif
Ce travail consiste à automatiser, via PowerShell :
- La création d’un dossier partagé.
- La création d’un groupe et d’utilisateurs Active Directory.
- Le mappage automatique d’un lecteur réseau à l’aide d’une GPO.
- L’activation du RDP pour un groupe d’utilisateurs.
- L’application de la GPO sur une OU spécifique.

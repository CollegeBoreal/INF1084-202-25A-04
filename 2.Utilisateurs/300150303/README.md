# Projet PowerShell AD - Étudiant 300150303

## Description
Ce projet a pour objectif de comprendre et de pratiquer la gestion des objets Active Directory (AD) à travers des scripts PowerShell simulés.  
Il inclut la création, la manipulation et l’exportation d’utilisateurs et de groupes.

---

## Contenu du dossier

| Script | Description |
|--------|-------------|
| `utilisateurs1.ps1` | Création d’utilisateurs simulés et ajout de nouveaux utilisateurs. |
| `utilisateurs2.ps1` | Création de groupes simulés et ajout d’utilisateurs dans les groupes. |
| `utilisateurs3.ps1` | Requêtes et filtres sur les utilisateurs (ex: prénom contenant "a"). |
| `utilisateurs4.ps1` | Export et import CSV des utilisateurs et création d’un groupe à partir du CSV. |
| `mini_projet.ps1` | Simulation complète : création d’utilisateurs Promo2025, groupe Etudiants2025 et export CSV. |
